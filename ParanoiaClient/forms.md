# Формы для клиента.

## Hello form
Первый вход, никаких данных нет 


Выбор:
- Подключиться к серверу
- Установить свой сервер


## Установить свой сервер
Input: Данные хоста Домен (для TLS, должен указывать на ip),ip, username, password, (порт для сервера)
Button: Установить.
Backend на вермя выполнения включить progress_bar:
- Создать админскую пару ключей admin_pub, admin_priv
- Подключение по ssh к хосту
- Создать /opt/Paranoia,  /opt/Paranoia/configs/Paranoia.json
- Внутри Paranoia.json: 
{
  "port": 1455,
  "store_path": "store",
  "admin_key": "admin_pub",
  "users": { }
}
- Установить nginx.
- Получить сертификат на домен, если его нет.
- Под этот домен сделать конфиг для nginx с терминацией на сервер Paranoia
- Запустить nginx
- Скачать последнюю версию paranoia-server с gitlab/github и т.д.
- Добавить сервис systemd для paranoia-server
- Запустить сервер
- Выход с хоста
- Сделать тестовый запрос на сервер, чтобы убедиться, что он поднялся Отправить REG запрос с пустым клиентом, убедиться, что (Bad pub_key:), а не Bad admin_sig
- Добавить сервер в список серверов клиента
- Перейти в основной раздел


## Подключиться к серверу
Выбор: 
- Админ
- Клиент


## Подключиться к серверу Админ
Input: Домен:port, admin_priv
Backend:
- Сделать тестовый запрос на сервер, чтобы убедиться, что он поднялся Отправить REG запрос с пустым клиентом, убедиться, что (Bad pub_key:), а не Bad admin_sig
- Добавить сервер в список серверов клиента
- Перейти в основной раздел


## Подключиться к серверу Клиент
Выбор: 
- Регистрация
- Вход


## Подключиться к серверу Клиент регистрация
- Сформировать пару ключей USER_PUB USER_PRIV
- Выдать в UI : USER_PUB с текстом: Передайте этот ключ администратору сервера, чтобы он вас зарегистрировал (можно qr code)
- после подтверждения, к форме Подключиться к серверу Клиент вход c автозаполнением поля private_key
 
## Подключиться к серверу Клиент вход
Input: Домен:port private_key(USER)
- Проверка входа через запрос на PULL. Если (Bad sig encoding), то клиент не зарегистрирован.
- Если зарегистрирован - на главную


## Главная
Вертикальное окно со вкладками Админ, User, Add. Отображаются вкладки, если есть доступы для них.
### User
- Дерево серверов (treeview)
- В каждом дереве можно добавить диалог. 
- Для этого будет кнопка +. При добавлении будет предлагаться отсканировать qr код собеседника или выдать свой. 
- После добавления - оба автоматически отправляют друг другу сообщение HI


### Admin
- Список серверов
- Для каждого есть кнопка, открывающая окно для регистрации пользователя (user)
